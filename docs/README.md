# FaultFinder ドキュメント

このディレクトリには FaultFinder の各種ドキュメントが含まれます。

## ディレクトリ構成

- `manual/` - ユーザーマニュアル、使い方ガイド
- `api/` - API 仕様、インターフェース定義
- `design-notes/` - 設計議論、技術的判断の記録（議事録）

## design-notes について

設計議論や技術的判断の記録を時系列で保存します。
正式なドキュメントに昇格する前の議論や、
後から振り返るための判断理由などを記録します。

ファイル名形式: `YYYY-MM-DD-topic-name.md`
